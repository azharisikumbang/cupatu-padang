# CUPATU PADANG

## Persiapan 

* Install PHP 8.2
* NodeJS 21
* NPM 10.2

## Instalasi

run front end dev
```bash
npm run install
npm run dev
```

run server
```bash
php artisan key:generate
php artisan migrate
php artisan db:seed
php artisan storage:link
php artisan serve
```
